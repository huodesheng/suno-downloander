# sunoDownloander

#### 介绍
suno和udio 下载的小工具。支持mp3和mp4下载
suno 和 udio 是近期比较火的 AI制作音乐的网站。官网虽然提供下载功能，但是要么步骤繁琐，要么无法下载其他人的。
我只做了这样一个网站。只要有分享链接，就可进行音乐的下载。MP3和MP4都可以。

[SunoSong下载，点击体验](https://sunoai.fun/)

[UdioSong下载，点击体验](https://sunoai.fun/udio)
